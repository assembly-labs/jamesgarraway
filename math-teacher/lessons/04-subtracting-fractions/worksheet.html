üìù Answer Key

1. 4/5 - 1/5 = 3/5

2. 6/8 - 2/8 = 4/8 (or 1/2 if simplified)

3. 5/6 - 3/6 = 2/6 (or 1/3 if simplified)

4. 3/4 - 1/4 = 2/4 (or 1/2 if simplified)

5. 7/10 - 4/10 = 3/10

6. 7/8 - 3/8 = 4/8 (or 1/2 if simplified)

7. 5/6 - 2/6 = 3/6 (or 1/2 if simplified)

8. 7/10 - 3/10 = 4/10 of the field = 40 yards

9. 10/10 - 8/10 = 2/10 (or 1/5 if simplified)

10. 5/8 - 3/8 = 2/8 (or 1/4) more pizza

11. 5/6 - 2/6 = 3/6 (or 1/2) cup more flour needed

12. Similar: Both keep the denominator the same; both work with the numerators only; both require same denominators. Different: Addition combines/adds pieces together; subtraction takes pieces away/removes them. Addition uses + and subtraction uses -.
