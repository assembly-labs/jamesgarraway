11. a) already simplified b) 3/5 c) already simplified d) 1/3

12. No, Emma is not correct. Simplifying does NOT change the amount - it just gives a simpler name for the same amount. 6/8 and 3/4 represent exactly the same amount, just like "twelve inches" and "one foot" are different names for the same length. When we simplify, we divide both numerator and denominator by the same number, which keeps the fraction equivalent.
